## 1. 引言
为什么会谈到“如何判断两条线段是否相交”这个问题呢？

当初调研了一些算法，找到了一个相对完善的版本，但最近推导下来发现还是不够严谨。
很有意思的是，我目前所使用的代码版本其实是源自于《Windows游戏编程大师的技巧》这本书的改进版本，其采用一种比较取巧的方式解决这个问题，
但由于考察之后发现不够严谨，所以，这里将这个问题旧事重提，准备通过数学原理和代码实现两个方面来讨论这个问题。
